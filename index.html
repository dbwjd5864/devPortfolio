YumPlan

Description 다른 사람들의 글로부터 식단 아이디어를 얻을 수 있고 내 플래너를 공유 함으로서 다른 사람들에게 내 아이디어를 공유할 수 있게 만들어주는 서비스입니다. 사이트의 UI, UX, 서비스, 기능들을 구상부터 시작하여 완전한 반응 형 웹사이트로 만들어내기까지 처음으로 React에 Redux를 더하여 사용하게 됐습니다.

A single page application showing the Meal Recipes built with MERN stack. User can get some meal ideas from other users. Users can post their weekly meal plan to share or create meal planner after login as well.

Language MongoDB Express React (React Hooks and Reducers) Node JS

Sweets

Description 키보드 좌,우 키를 이용하여 캐릭터를 컨트롤하며 떨어지는 호박을 피해 사탕을 모으는 자바스크립트 캔버스를 이용한 간단한 게임 입니다. 오직 기본 HTML, CSS, JavaScript Canvas만을 이용하여 게 임을 구현해보면서 JavaScript 자체에 포커스를 높였습니다.

A simple game built with javascript and canvas (School Project). The player controls their character using keyboard’s left and right arrow keys to collect the falling candy avoiding the falling pumpkins.

Language Html Javascript Css

How Is Your Day?

Description 나만의 To-Do 리스트를 작성 할 수 있고 매번 바 뀌는 사진들과 인용문들이 사용자에게 영감을 줄 수 있게끔 디자인하고 구현해 보았습니다. 처음으로 Webpack을 더함으로써 애플리케이션 모듈을 정적인 소스들로 생산해 보았습니다. 그 과정 속에서 각 모듈이 어떻게 하나의 bundler로 모아지며 여러 파일들의 관리를 용이하게 만드는지에 대해 알아볼 수 있었고 브라우저에서의 파일 로딩 시간을 줄여 Frontend와 backend 사이에 소통을 더 원활이 할 수 있다는 것을 느낄 수 있었습니다.

"How is your day" is a full-stack web application built with the MERN stack and webpack. User can add a to-do list and write feelings about the day to dear diary. The Quote of the day is also shown under dear Diary section.

Language MongoDB Express React Node JS Webpack
